HOW TO USE
================================================================================

START DEVELOPMENT
  $ pnpm dev
  Visit: http://localhost:3000

CREATE ACCOUNT
  1. Click "Sign Up" button
  2. Fill 5-step form
  3. Verify email
  4. Login to account

BROWSE PROFILES
  1. Go to /browse
  2. Swipe right (drag) to like
  3. Swipe left (drag) to pass
  4. Or use heart/pass buttons
  5. View matches in /connections

VIEW PROFILE
  1. Go to /your-profile
  2. See all your information
  3. Click /edit-profile to update

MANAGE CONNECTIONS
  1. Go to /connections
  2. See sent likes
  3. See received likes
  4. Accept/reject requests
