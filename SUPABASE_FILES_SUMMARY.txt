================================================================================
  TROUBLESHOOTING
================================================================================

Problem: Environment variables not loading
  → Delete/clear app cache
  → Restart npm run dev
  → Verify .env.local file format (no spaces around =)

Problem: Can't connect to Supabase
  → Check internet connection
  → Verify Supabase project is active
  → Check URL format (https://xxxx.supabase.co)

Problem: Invalid API key error
  → Copy keys again from Supabase Settings → API
  → Avoid extra spaces
  → Restart dev server

Problem: Tables not created in database
  → Run migration SQL again
  → Check SQL execution output for errors
  → Verify in Supabase Table Editor

Problem: Admin login fails
  → Verify admin user was created in database
  → Check email and password match
  → Query database: SELECT * FROM users WHERE role = 'admin'

For detailed troubleshooting:
  → See SUPABASE_SETUP_GUIDE.md (Troubleshooting section)
